# 密排六方金属
1. 与其他金属一样，多晶 cph 金属的变形始于滑移，但晶格的对称性和滑移系统的可用性低于立方金属。在这些条件下，滑移不能继续作为唯一的变形模式，孪生在相当低的应变（“<0.2”）下变得明显。
2. 由于孪生剪切值较小，cph 金属中的孪生比立方体金属中的孪生更宽。
